# Git
Download [git](https://git-scm.com/downloads)

Git commands:

```git
$ git status
```
```js
$ git add <file>
```
```js
$ git commit -m "<message>"
```
```js
$ git push <to some branch>
```
```js
$ git pull <from some branch>
```
